# Report Feedback Automation

## 概要

Notionで記録している日報を対象に、夜間バッチでAIフィードバックを自動生成し、
翌日の振り返りと行動改善につなげるための**日報自動フィードバックツール**です。

フリーランスのシステムエンジニアとして、
「日報を書いて終わり」「振り返りが形骸化する」状態を避けるために構築しました。

---

## 背景・課題

- 日々の業務内容はNotionに日報として記録している
- ただし、以下の課題を感じていた
  - 振り返りが主観的になりがち
  - 学びや改善点が流れてしまう
  - 忙しい日は振り返りを省略してしまう
- 人にレビューしてもらうのはコストが高く、継続が難しい

---

## 解決したこと

- 日報を**人が読まなくても振り返れる仕組み**を構築
- 夜間にAIが日報を読み取り、メンター視点でフィードバックを生成
- 状態管理（未処理 / 処理済み）を明確にし、運用負荷を最小化

---

## 特徴

- Notionを中心にした既存運用に自然に組み込める
- 平日夜間に1回だけ動く軽量バッチ
- 無料枠前提（Gemini API × AWS サーバレス）
- 冪等性を考慮した安全な再実行設計
- 実務に即したフィードバック形式（要約・良かった点・改善点・次の一手）

---

## 想定ユーザー

- 日報や作業ログを継続しているエンジニア
- フリーランス / 個人開発者
- 自分の業務を客観視し、改善サイクルを回したい人
